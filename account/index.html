Что может администратор:

Чтобы стать администратором, заполните форму ниже.

  • Создавать опросы;
  • Смотреть статистику прохождения опросов;
  • Создавать специальные артикулы для опросов;
  • Назначать людей для прохождения опроса;
  • Назначать людей для ЕСО.
Если возникли вопросы, пишите.

Для чего нужна эта платформа?

Эта платформа предназначена для создания администраторами опросов, пользователи в свою очередь могу проходить эти опросы.

Как пройти опрос?

Вы можете пройти опрос на главной странице или во вкладке “Опросы”. На главной странице в нужном поле необходимо ввести артикул опроса и нажать на кнопку «Пройти». Во вкладке “Опросы” можно либо ввести артикул, либо пройти опрос, который назначен администратором. Назначенный опрос будет находится в списке «Новые опросы».

Что может пользователь?

Пользователь имеет следующие возможности:

  • Проходить опросы (как по артикулу, так и назначенные администратором);
  • Смотреть свои результаты прохождений;
  • Подать заявку на получения статуса «Администратор».

Что может администратор?

Администратор имеет следующие возможности:

  • Создавать опросы;
  • Смотреть статистику прохождения опросов;
  • Создавать специальные артикулы для опросов;
  • Назначать людей для прохождения опроса;
  • Назначать людей для ЕСО.

Как стать администратором?

Для того, чтобы стать администратором, Вам следует заполнить заявку в разделе “Личный кабинет” “Администратор”. В ней нужно в обязательном порядке написать причину, по которой Вы хотите стать администратором. Заявка будет рассмотрена в течение 7 дней, после чего Вы получите ответ.

Что такое артикул опроса и как его получить?

Артикул опроса – шестизначный числовой идентификатор опроса, который создается автоматически системой. Администраторы могут создавать специальные артикулы для своих опросов (если они не заняты). Получить артикул можно лично у администратора. Просмотреть артикул опроса можно в параметрах опроса.

Что такое ЕСО и для чего оно нужно?

Единоразовое создание опроса (ЕСО) – возможность пользователя создания одного опроса совместно с администратором. Чтобы получить эту возможность, администратор должен выдать пользователю (по его ID) разрешение. ЕСО служит для создания совместных опросов (к примеру, если администратор не может сам создать опрос, то он может выдать разрешение пользователю, чтобы он помог ему). Просмотр результатов опроса будет доступен только администратору.

Как назначить людей для ЕСО?

Для того, чтобы назначить людей для ЕСО, нужно во время создания опроса зайти во вкладку “ЕСО” “Добавить пользователя” и ввести ID пользователя. Пользователь, в свою очередь, должен принять приглашение.
